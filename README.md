# Newsbooze
`NewsBooze` is a simple CLI tool built using Node which displays the top news headlines from Times of India and Tech Crunch 
right in the terminal. This is a handy tool which gives you a quick doze of headlines without getting distracted by your smartphone
and brings you the best of both National and Tech News.
The news is fetched using [News API](https://newsapi.org/)
This package is published at https://www.npmjs.com/package/newsbooze

## Dependencies
* [ora](https://github.com/sindresorhus/ora)
* [axios](https://www.npmjs.com/package/axios)
* [minimist](https://github.com/substack/minimist)

## How to use
* Install the package by `npm install newsbooze`
* Run the command `newsbooze [option]` for Displaying the news on the terminal by a source of your choice.

#### Options 
* `techcrunch` : Tech Crunch
* `toi` : Times of India
